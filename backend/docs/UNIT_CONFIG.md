## UnitConfig

Справочник юнитов. Используется как шаблон при создании юнитов в игре.

### Назначение
Хранит базовые характеристики юнитов, которые улучшаются при повышении уровня.

### Поля

- id — уникальный идентификатор
- name — уникальное имя юнита
- hp — базовое здоровье
- attack — базовый урон
- attack_speed — скорость атаки (время в секундах между ударами)
- attack_range — дальность атаки (в клетках)
- attack_type — тип атаки (melee / ranged. Если melee, то attack_range=1, ranged — attack_range>1)
- crit_chance — шанс критического удара (0–1)
- crit_damage — множитель критического урона
- move_speed — скорость перемещения (клетки/секунду)
- mana_cost — стоимость покупки
